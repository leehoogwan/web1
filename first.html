HTML

→definition(정의)
The HyperText Markup Language, or HTML is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

→사용법
●"br tag"
"br"tag를 사용하면 줄바꿈 효과가 나타난다.(다음줄로 넘어갈 수 있음.)
If we use "br" tag, we can go to next line.
●이미지를 넣는 방법

●"p tag"
"p"tag 는 단락을 나누는 역할을 한다. 그래서 그냥 다음 줄로 넘어가는게 아닌 단락변경은 "p"tag내에서 해결하면 된다.
If you want to set or change the paragragh, then you can use "p" tag.
